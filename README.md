# Logistics Performance Intelligence Hub

Projeto de portfólio focado em análise logística, BI e automação de processos.

## 🎯 Objetivo
Simular uma operação logística completa, com foco em:
- KPIs operacionais
- Redução de custos
- Eficiência de entregas
- Automação de alertas

## 🧱 Stack
- SQL (PostgreSQL)
- Power BI
- n8n
- Grafana / Looker
- Python (ETL)

## 📊 Entregáveis
- Base de dados simulada
- Modelagem dimensional
- Dashboards operacionais
- Alertas automatizados
- Insights de negócio
