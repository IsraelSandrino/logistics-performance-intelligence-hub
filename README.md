# Logistics Performance Intelligence Hub

Projeto de portfólio focado em análise logística, BI e automação de processos.

## 🎯 Objetivo
Simular uma operação logística completa, com foco em:
- KPIs operacionais
- Redução de custos
- Eficiência de entregas
- Automação de alertas

## 🧱 Stack
- SQL (PostgreSQL)
- Power BI
- n8n
- Grafana / Looker
- Python (ETL)

## 📊 Entregáveis
- Base de dados simulada
- Modelagem dimensional
- Dashboards operacionais
- Alertas automatizados
- Insights de negócio

## Arquitetura da Solução

![Arquitetura da Solução](/assets/logistics-solution-architecture.svg)

O projeto segue um fluxo simples e objetivo:

```txt
Dados fictícios gerados por Python
        ↓
PostgreSQL
        ↓
Power BI
        ↓
n8n
```

A arquitetura foi definida para simular uma operação logística orientada por dados, com armazenamento estruturado, análise de KPIs em BI e automações/alertas operacionais.
